# Human Vital Signs ML Project

## Overview
This project demonstrates a step-by-step Python & Machine Learning workflow to analyze and predict **Heart Rate** based on human vital signs such as Age and Steps. The project includes:

- Data loading and checking
- Data cleaning and preprocessing
- Data visualization
- Machine learning model training (Linear Regression)
- Model evaluation and result visualization
